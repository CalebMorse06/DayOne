# DayOne — Knowledge Extraction & Digital Twin Engine

Transform undocumented industrial expertise into structured, verifiable, and spatially-aware learning modules.

## 🚀 One-Click Quick Start
1. **Initialize:** `./setup.sh`
2. **Keys:** Add your `GEMINI_API_KEY` to `.env.local`
3. **Launch:** `npm run dev`

---

## 🛠 Business & Technical Foundation

### 1. The Knowledge Pipeline
- **Tacit Extraction:** Gemini 2.0 Flash pulls expert "vibes" (sounds, feelings) from video.
- **Spatial Mapping:** Automatic 3D coordinate generation for parts.
- **SOP Automation:** One-click Standard Operating Procedure generation.

### 2. Production Infrastructure
- **Auth & Multitenancy:** Scalable organization support via Supabase.
- **Revenue Engine:** Stripe Checkout & Customer Portal integration.
- **Compliance:** Verified Certification system with data integrity hashing.
- **Mobile Native:** Capacitor-ready for iOS/Android distribution.

### 3. Usage & Paywalls
- **Free Tier:** 2 courses, 30 min video, 5 AI questions/course.
- **Hard Gates:** Automated enforcement in `lib/paywall.ts`.

---

## 🏗 Developer Roadmap
- **Deployment:** Connect to Vercel and add environment variables.
- **Database:** Run SQL schema in `lib/supabase.ts`.
- **Mobile:** Run `npx cap open ios` to launch Xcode.
